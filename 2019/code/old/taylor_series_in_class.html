Let's expand $e^x$ out to five terms. We'll do this iteratively where we will use the $0$th order approximation to get the $1$st order approximation, and so on.

In [2]:
# max number of terms we want to expand
n_terms = 5

# range of x values to look at
x_vals = np.linspace(0,2,100)

# intialize the array of y_vals
y_vals = np.zeros([n_terms,len(x_vals)])

# loop through the number of terms
for n in range(n_terms):
    
    # special case the 0th order
    if n == 0:
        y_vals[0,:] = np.ones(len(x_vals))
      
    # otherwise nth order is n-1th order plus new term
    else:
        new_term = x_vals**n / math.factorial(n)
        y_vals[n,:] = y_vals[n-1,:] + new_term

Now let's plot the results of our approximations and compare to $e^x$.
